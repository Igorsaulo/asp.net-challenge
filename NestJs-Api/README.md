# NestJs-Api

## Description
<p>Este projeto foi desenvolvido em Nestjs, com o objetivo de criar uma API analisar e extrair dados de um de um aruivo de Video </p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
